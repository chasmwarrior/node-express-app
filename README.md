# Node Express App

This is a simple Node.js and Express application.

## Cara Menjalankan

1. Clone repositori ini:
https://github.com/chasmwarrior/node-express-app.git

2. Masuk ke direktori proyek:
```bash
cd node-express-app

3. Instal dependensi:
npm install

4. Jalankan aplikasi:
npm start
